# Security Policy

## Reporting a Vulnerability

Found a security issue in this codebase or on [hasimuener.de](https://hasimuener.de)?

**Please report responsibly** — do not open a public GitHub issue.

📧 **Contact:** `hallo@hasimuener.de`

### What to include

- Short description of the vulnerability
- Steps to reproduce (URL, request, payload)
- Impact assessment (data, access, integrity)
- Your preferred disclosure timeline

### What you can expect

- **Acknowledgement** within 48 hours (working days)
- **Initial assessment** within 5 working days
- **Fix or mitigation timeline** depending on severity
- Credit in the changelog if you wish

### Scope

In scope:
- This repository (theme, automation, REST endpoints)
- Live site `hasimuener.de` and direct subdomains
- Server-side GTM container endpoints

Out of scope:
- Third-party services (Cal.com, GTM/GA4, Brevo, n8n Cloud)
- DoS / volumetric tests
- Social-engineering attacks

Thank you for keeping the site and its users safe.
